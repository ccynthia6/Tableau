## Tableau Project
Built a dashboard using tableau.

## Dataset Used
[Dataset](https://www.kaggle.com/datasets/alexanderfreberg/airbnb-listings-2016-dataset/data)

## Dataset Description
The dataset is an excel file.  
The dataset has details about airbnb. The dataset has three tabs Calendaer, Listings and Reviews.
Listing: Contains the details of all the airbnb's in Seattle, Washington, USA, such as the lisiting id, name of the airbnb, description, location, neighbourhood, city, number of rooms, price, cleaning fee, reviews per month and so on.
Calendar: Contains the availability of the airbnb, price date and listing id.
Reviews: Contains the listing id, review id, reviewer's name and the commnet/review.
